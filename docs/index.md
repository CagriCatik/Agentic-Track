## What This Documentation Covers

- Foundations: prompts, chains, models, function calling, and core agent patterns.
- Build systems: RAG pipelines, LangGraph flows, reflection/reflexion loops, and MCP integrations.
- Production concerns: evaluation, debugging, tracing, security, and operational tradeoffs.

## How To Use It

1. Open a chapter `Overview` first to understand scope and expected outcomes.
2. Expand `Lessons` in the sidebar and work through pages in order.
3. Capture your own notes, code snippets, and Mermaid diagrams directly on lesson pages.
4. Use the glossary and security chapters as reference while building real projects.
